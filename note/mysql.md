# 小册笔记

## `MYSQL`的客户端/服务器架构

使用过程

1. 用户通过客户端发送请求
2. 从客户端发送到服务器，可多个客户端
3. 服务器响应请求

`mysql`服务器程序的进程也被称为`mysql数据库实例`简称数据库实例

每个进程都有一个唯一的编号，称为`进程ID`(`PID`)

1. 启动程序时由操作系统随机分配
2. id不重复
3. id可回收

每个进程都有一个名称

1. 用户自定义，有默认名称

`mysql`服务器程序

`tcp`/`ip`默认`port:3306`

```
mysqld -P3307(端口号)
```

Windows使用`命名管道`或`共享内存`
